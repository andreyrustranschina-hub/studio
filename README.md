# Local Video Manager

Простое и интуитивно понятное приложение для управления вашими локальными видеофайлами прямо в браузере. Этот инструмент использует File System Access API, чтобы позволить вам сканировать каталоги, переименовывать файлы и организовывать вашу видеотеку без загрузки чего-либо в облако.

![Скриншот приложения](placeholder.jpg "Интерфейс Local Video Manager")

## Возможности

-   **Сканирование локальных каталогов**: Выберите любую папку или целый диск на вашем компьютере для поиска видеофайлов (`.mp4`, `.mov`, `.mkv`, `.avi`, `.webm`).
-   **Рекурсивный поиск**: Приложение автоматически находит видео во всех подкаталогах выбранной папки.
-   **Мгновенная обратная связь**: Видео появляются на экране сразу после их обнаружения, вам не нужно ждать завершения сканирования.
-   **Переименование файлов**: Переименовывайте файлы прямо на вашем диске с помощью удобных кнопок.
-   **Исключение папок**: Легко исключайте папки из сканирования прямо из карточки видео.
-   **Встроенный плеер**: Воспроизводите видео, кликнув по нему, во встроенном модальном окне.
-   **Ленивая загрузка**: Оптимизированная производительность для больших библиотек — видео подгружаются по мере прокрутки.
-   **Генерация миниатюр**: Автоматическое создание миниатюр для предварительного просмотра видео.

## Как запустить

1.  **Установите зависимости**:
    ```bash
    npm install
    ```
2.  **Запустите сервер для разработки**:
    ```bash
    npm run dev
    ```
3.  Откройте `http://localhost:9002` в вашем браузере (Google Chrome или другой браузер, поддерживающий File System Access API).
